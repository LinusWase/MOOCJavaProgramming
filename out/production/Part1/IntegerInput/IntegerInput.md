# Integer Input

Write a program that asks the user for a value. The program should then print the value provided by the user.

Here's a couple of examples:
### Sample output

>Give a number: <br>
>3 <br>
>You gave the number 3 <br>
 
### Sample output

>Give a number: <br>
>42 <br>
>You gave the number 42 <br>
